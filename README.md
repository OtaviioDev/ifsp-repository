# IFSP Academic Repository

A centralized platform developed to catalog, preserve, and showcase academic productions created by students at IFSP (Federal Institute of São Paulo). This project was originally conceived and built during the **IFSP Jacareí Hackathon**.

## 🎯 Objective

The platform addresses a real-world community challenge by centralizing institutional knowledge that is usually scattered across departments. It allows students and faculty to easily upload and discover:
- **Theses (TCCs)** and graduation papers.
- **Scientific articles** and research essays.
- **Prototypes** and innovative student projects.
- **Coursework** and shared open educational resources.

## 🛠️ Tech Stack & Architecture

- **Structure**: Semantic HTML5 for web accessibility.
- **Styling**: Responsive CSS3 for seamless cross-device layouts.
- **Logic**: Vanilla JavaScript (ES6+) powering dynamic search, user interactions, and DOM manipulation.

## 🚀 Current Features (Functional Prototype)

- **Dynamic Search Engine**: Real-time project filtering via keyword queries.
- **Course-Based Indexing**: Automatic categorization based on specific academic fields and departments.

## 💻 Hackathon Development & AI Integration

This functional MVP was built as a collective team effort within a high-pressure, time-constrained hackathon environment. 

* **Our Process**: To accelerate development speed, research core features, and bootstrap the software architecture under tight deadlines, our team integrated advanced Artificial Intelligence tools, including **DeepSeek**. 
* **Quality Assurance**: Every AI-generated code snippet was strictly reviewed, manually refactored, and tested by the team to meet performance standards and maintain architecture control.
